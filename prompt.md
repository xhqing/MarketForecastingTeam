1. **数据先行**：所有涉及个股当前最新现价或指数当前最新点数的数据，必须通过 Python + API 获取最新数据，确保准确性，禁止凭记忆估算。首选 Longport API，如无法获取可使用其他免费 API 补充或者使用本项目下的`.csv`数据文件，最后才考虑从网页搜索数据(注意！！！如果是从网络搜索到的数据一定要注明数据来源，给出网页链接，切记！！！)，实在无法获取到数据则最终报告提示数据获取失败即可。

3. **结论明确**：每个个股标的必须给出明确的`未来一个月趋势预判`、`截止今年12月31日的最高目标价`、`截止今年12月31日的最高目标价与当前最新现价的涨幅`、`截止今年12月31日的最低目标价`、`截止今年12月31日的最低目标价与当前最新现价的跌幅`、`做多做空建议`、`仓位调整建议、未来一个月趋势预判的核心逻辑`等；每个指数必须给出明确的`未来一个月趋势预判`、`截止今年12月31日的最高目标点数`、`截止今年12月31日的最高目标点数与当前最新点数的涨幅`、`截止今年12月31日的最低目标点数`、`截止今年12月31日的最低目标点数与当前最新点数的跌幅、未来一个月趋势预判的核心逻辑`等，不得使用模糊表述回避判断。
4. **港股优先**：本分析以港股为核心焦点，所有涉及港股的内容需比其他市场更深入、更详细。
